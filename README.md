## 🌟 這是什麼？ (淺顯介紹)

想像你站在一個由方格組成的房間裡。你的目標是走到「終點 🔴」，並在途中避開危險的「障礙物 🧱」。每一次移動，你都可以選擇「上、下、左、右」。

如果是我們自己走，可能要試走很多次、甚至走進死胡同。但這個小程式展示了 **AI 如何透過「強化學習 (Reinforcement Learning)」的超能力**，瞬間看透整個地圖的結構，幫你規劃出一條不會繞遠路、且能安全避開障礙物的高智商完美路徑！

## 🚀 遊玩指南 (如何使用)

只要跟著直覺的介面點擊，你也能輕鬆成為 AI 訓練師！

1. **📏 決定房間大小**：在左上角選擇 5x5 到 9x9 的網格大小。
2. **🟢 放置起點**：在網格上隨便點擊一下，設定出發點。
3. **🔴 放置終點**：再點擊一下，設定你要抵達的目的地！
4. **🧱 放置障礙物**：最後點幾下，放上一些不能撞到的牆壁。
5. **🎲 嘗試隨機亂走 (Generate Policy)**：讓電腦在每個格子裡隨機決定方向，看看會發生什麼事。
6. **✨ 評估路線好壞 (Evaluate Policy)**：程式會幫所有格子「打分數」！綠色代表這個位置「離終點很近、很有希望」，紅色代表「很糟糕」。
7. **👑 找出完美解答 (Find Optimal Policy)**：見證奇蹟的時刻！按下王冠按鈕，系統會瞬間算出最佳解，並用 **翡翠色的發光特效與脈衝動畫** 幫你指引出一條最完美的尋寶路線！
